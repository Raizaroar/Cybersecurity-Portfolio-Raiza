# Assets Directory

This directory contains all media resources used across the portfolio.

## Structure

- **images/**: General portfolio images
  - **screenshots/**: Project screenshots organized by lab
  - **badges/**: Certification and skill badges
  - **banners/**: Header images and banners
- **diagrams/**: Network diagrams, attack flows, and technical illustrations

## Usage

Reference images in markdown using relative paths:
```markdown
![Alt text](../assets/images/filename.png)
```

## Naming Convention

Use descriptive, lowercase names with hyphens:
- ✅ `nmap-scan-results.png`
- ✅ `burp-suite-intercept-request.png`
- ❌ `image1.png`
- ❌ `Screenshot 2024.png`
